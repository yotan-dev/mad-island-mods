## Special Variables

### actors[{index}].tits
The tit size of the actor at the given index. Formatted in "00" format. ("00", "01", "02", etc.)

### actors[{index}].disleg
Prefix for characters missing legs ("DisLeg_") if the actor at the given index has missing legs, empty string otherwise.
